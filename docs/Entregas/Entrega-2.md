# Entrega 2

## Introdução

A entrega 2 consistiu na elicitação e priorização de requisitos, além do desenvolvimento de artefatos como o perfil de usuário e a criação de personas. Durante essa etapa, também realizamos uma mudança significativa no projeto: o aplicativo escolhido para análise foi alterado de "e-Título" para "FGTS". Essa alteração impactou diretamente nossa documentação, onde atualizamos todos os artefatos importantes, refletindo o novo foco do projeto. Além disso, aplicamos técnicas de elicitação e priorização de requisitos, garantindo uma análise mais precisa e alinhada às necessidades do usuário.
